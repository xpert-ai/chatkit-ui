# @xpert-ai/chatkit-types

Types for the ChatKit WebComponent.

## Setup

Add `@xpert-ai/chatkit-types` to `compilerOptions.types` in your `tsconfig.json`. This registers the `xpert-ai-chatkit` element and its CustomEvents in the global DOM typings.

```json
{
  "compilerOptions": {
    "types": ["@xpert-ai/chatkit"]
  }
}
```
